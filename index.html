Diplomatura en Superpoderes Digitales

Curso 1 – Domina la Programación con la Mente

Semana 1: Introducción a la programación visual
Exploramos Tinkercad y ArduinoBlocks. Entendemos qué es una placa Arduino y cómo la programamos sin escribir código.
Semana 2: Primeros circuitos con luces
Aprendemos a encender y apagar LEDs virtuales en Tinkercad, usando bloques como si fueran piezas de LEGO.
Semana 3: Entrenando a ChatGPT como programador
Aprendemos a hablar con la IA para que genere código de Arduino. Instrucciones simples para resultados mágicos.
Semana 4: Laboratorio remoto - LEDs reales
Accedemos al laboratorio remoto y encendemos luces reales desde casa. ¡La magia ocurre a distancia!
Semana 5: Motores y movimiento
Controlamos motores pequeños. Aprendemos cómo hacer que giren y se detengan desde bloques o comandos de IA.
Semana 6: Sensores inteligentes
Incorporamos sensores de luz o movimiento. Detectamos el mundo real y reaccionamos con código inteligente.
Semana 7: Proyecto integrador
Creamos nuestro primer superproyecto: un sistema con luces, sensores y motores, todo controlado con bloques e IA.
Semana 8: Presentación del superpoder
Grabamos o presentamos nuestro proyecto. Obtenemos el diploma de programador mental con Arduino + IA.

Curso 2 – Crea Tableros de Comando con IA

Semana 1: ¿Qué es un tablero de comando?
Aprendemos la idea de un dashboard visual para controlar luces, motores o sensores desde la web.
Semana 2: Diseño de botones y sliders
Usamos IA para diseñar botones, interruptores, barras deslizantes y trackballs con estilo futurista.
Semana 3: Código HTML/CSS generado por IA
Aprendemos a pedirle a ChatGPT que genere el código web necesario para construir interfaces interactivas.
Semana 4: Comunicación con placas
Integramos la web con las placas. Pulsar un botón en la web enciende una luz en la vida real.
Semana 5: Estilos y personalización visual
Personalizamos el aspecto del tablero para que luzca profesional, moderno y único.
Semana 6: Automatización inteligente
Creamos tableros que no solo controlan, sino que toman decisiones con la ayuda de la IA.
Semana 7: Proyecto final
Construimos nuestro tablero maestro: una interfaz IA + web + hardware. ¡Nuestro centro de control de superhéroes!
Semana 8: Exhibición y diploma
Presentamos nuestros tableros, compartimos experiencias y obtenemos el diploma como creadores de interfaces inteligentes.

Curso 3 – Construye Servidores Web Embebidos

Semana 1: ¿Qué es un servidor embebido?
Descubrimos que nuestra placa puede transformarse en una página web viva, sin depender de plataformas externas.
Semana 2: Configuración inicial de ESP8266
Preparamos la placa para recibir código embebido que creará su propia web desde cero.
Semana 3: Creación del HTML embebido
Le enseñamos a la IA a generar una página web lista para ser almacenada en la placa y servida al mundo.
Semana 4: Control sin intermediarios
Comprobamos cómo podemos controlar luces, motores o sensores sin necesidad de plataformas externas.
Semana 5: Automatización directa
Nuestra placa toma decisiones automáticamente y actualiza su web de forma autónoma.
Semana 6: Personalización avanzada
Editamos el estilo y el diseño de nuestro servidor embebido para que tenga identidad única.
Semana 7: Proyecto total
Creamos nuestro servidor web final: diseño + código + embebido. ¡Una placa con superpoderes!
Semana 8: Graduación final
Presentamos nuestros servidores, y celebramos la obtención del diploma como Diplomado en Superpoderes Digitales.
